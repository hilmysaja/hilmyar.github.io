## 📂 Struktur Repositori
Proyek ini memiliki struktur direktori sebagai berikut:

* `about/` - Halaman yang berisi informasi detail mengenai profil dan latar belakang saya.
* `contact/` - Halaman untuk informasi kontak yang dapat dihubungi.
* `dokumen/` - Tempat penyimpanan berkas atau dokumen pendukung portofolio (seperti CV atau sertifikat).
* `img/` - Folder berisi seluruh aset gambar dan foto yang digunakan di website.
* `sosmed/` - Aset atau halaman yang menautkan ke profil media sosial saya.
* `index.html` - Halaman utama (beranda) dari website portofolio.

## 🛠️ Teknologi yang Digunakan
* **HTML** (100%) - Sebagai kerangka utama website.
